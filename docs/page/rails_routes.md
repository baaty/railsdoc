### 説明

定義されている全ルーティングを出力

### 使い方

    $ rails routes [オプション]

### オプション

| 名前                          | 説明                |
| --------------------------- | ----------------- |
| -c, --controller=CONTROLLER | 特定のコントローラでフィルタリング |
| -g, --grep=GREP             | 特定のパターンでルートを検索    |
| -h, --help                  | ヘルプ               |

### 例

    $ rails routes
