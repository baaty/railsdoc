### 説明

ログファイルなどに出力

### 使い方

    logger.ログレベル(値)

### ログレベル

| ログレベル  | 説明           |
| ----------- | -------------- |
| unknown(値) | 不明なエラー   |
| fatal(値)   | 致命的なエラー |
| error(値)   | エラー         |
| warn(値)    | 警告           |
| info(値)    | 通知           |
| debug(値)   | デバック情報   |

### 例

#### 致命的なエラー

    logger.fatal('致命的なエラーです')
