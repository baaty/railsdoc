### 説明

コードからコメントの部分を抽出

### 使い方

    $ rails notes [オプション]

### オプション

| 名前                              | 説明                   |
| ------------------------------- | -------------------- |
| -a, --annotations=one two three | 特定のアノテーションによるフィルタリング |
| -h, --help                      | ヘルプ                  |

### 例

    $ rails notes
