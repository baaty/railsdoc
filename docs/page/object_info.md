### クラスとインスタンス

- クラスはオブジェクトの設計図
- インスタンスは、設計図に基づいて作成されたオブジェクト

### オブジェクトの作成

    クラス.new

### メソッドの呼び出し制限

| レベル    | 説明                                                 |
| --------- | ---------------------------------------------------- |
| public    | メソッドはどこからでも呼び出せる                     |
| protected | 同じクラスやサブクラス内のメソッドの中だけ呼び出せる |
| private   | 同じクラスやサブクラス内のメソッドの中だけ呼び出せる |
