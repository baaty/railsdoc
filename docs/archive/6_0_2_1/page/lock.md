あらかじめカラムのロックバージョンを記録しておき、更新時にロックバージョンが変わっていないことをバリデーションして保存

      end

      end
    end

### 悲観的ロック
データベース側でレコードをロックし、並行に更新できないようにする

    locked_entry = Entry.find(１，lock: true)

* MySQLとPostgreSQLでのみ利用可能
