### 説明
データベースのカラムをバージョンアップ  
upメソッドはロールバックできない処理をする際に主に使用

### 使い方
    def up
    end

### 例
#### pagesテーブルにnameカラムをstring型で作成
    def up
      create_table :pages do |t|
        t.string :name
        t.string :category
      end
    end
