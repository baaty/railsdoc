### 説明
フラグメントキャッシュを破棄

### 使い方
    expire_fragment(キャッシュキー [,  オプション])

### キャッシュキー
キャッシュキーは次の3つの形式
* 文字列
* ハッシュ
* 正規表現

### 例
#### 文字列
    expire_fragment "detail"

#### ハッシュ
    expire_fragment :controller => 'posts', :action => 'show', :id => post.id

#### 正規表現
    expire_fragment %r{/*/user/home}
