### 説明
保存される前に変更された値をハッシュで取得

### 使い方
    モデル.previous_changes()

### 例
#### 保存される前に変更された値をハッシュで取得
    person.name # "bob"
    person.name = 'robert'
    person.save
    person.previous_changes # {name: ["bob", "robert"]}
