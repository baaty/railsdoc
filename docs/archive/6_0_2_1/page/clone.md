### 説明
モデルのコピーを生成

### 使い方
    モデル.clone()

### 例
#### モデルのコピーを生成
    user = User.first
    new_user = user.clone
    user.name               # "Bob"
    new_user.name = "Joe"
    user.name               # "Joe"
