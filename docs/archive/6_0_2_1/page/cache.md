### 説明
ページ全体ではなく、ページの一部をキャッシュするときに使う

### 使い方
    <% cache([キャッシュキー, オプション]) do %>
      キャッシュ
    <% end %>

### オプション

オプション     | 説明
------------ | --
:skip_digest | digestの付与をスキップ

### 例
#### ページの一部をキャッシュ
    <% cache [ project, current_user ] do %>
      <b>All the topics on this project</b>
      <%= render project.topics %>
    <% end %>
